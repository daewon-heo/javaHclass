*8번 문제

자동차 속도조절 프로그램
입력된 속도에 따라 자동차 속도를 조절하는 프로그램 작성
100km/h 이상으로 달리면 "과속입니다. 속도를 줄이세요" 출력
음수를 입력하면 속도를 줄인다.
주행 후 0km/h 이하로 가면 "정지합니다" 출력후 프로그램 종료

ex)

출발 대기.....

속도 입력 : 10
10km/h 의 속도로 주행합니다.

속도 입력 : 30
40km/h 의 속도로 주행합니다.

속도 입력 : 70
110km/h의 속도로 주행합니다.
과속입니다. 속도를 줄이세요

속도 입력 : -80
30km/h의 속도로 주행합니다.

속도 입력 : -50
0km/h 속도로 주행합니다.
정지합니다.

   //현재채널에서 한칸 아래로 이동
 };

 chMove(int num){
  //입력된 번호로 채널 이동
 };

 volUp(){
  // 볼륨을 +1 조절
 };

 volDown(){
  // 볼륨을  -1 조절
